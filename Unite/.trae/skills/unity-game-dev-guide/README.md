# Unity游戏开发指南 Skill 使用说明

## 🎯 技能介绍

Unity游戏开发指南是一个专注于Unity和C#游戏开发的智能助手，为开发者提供专业、全面的技术支持和学习资源。

### 核心功能

1. **技术问题解答**：详细解析游戏开发中遇到的技术问题
2. **代码示例提供**：提供实用的代码示例和实现方案
3. **学习路径规划**：提供系统化的学习建议和进阶指南
4. **性能优化指导**：提供游戏性能优化的最佳实践
5. **技术领域覆盖**：涵盖从基础到高级的游戏开发技术
6. **问题保存管理**：将问题和回答保存为Markdown文件，方便后续查阅

## 🎓 学习路径

### 初级开发者
1. **基础编程**：C#基础、Unity编辑器使用
2. **核心系统**：输入系统、物理系统、UI系统
3. **游戏机制**：角色移动、碰撞检测、基本AI
4. **项目实践**：小型2D游戏开发

### 中级开发者
1. **高级编程**：设计模式、异步编程、内存管理
2. **性能优化**：批处理、对象池、Profiler使用
3. **系统设计**：游戏架构、状态管理、事件系统
4. **项目实践**：3D游戏开发、网络多人游戏

### 高级开发者
1. **前沿技术**：DOTS、ECS、Job System
2. **渲染优化**：高级着色器、自定义渲染管线
3. **工具链**：CI/CD、自动化测试、性能监控
4. **项目实践**：大型游戏项目、跨平台发布

## 🔍 如何使用

### 提问方式

您可以以自然语言的方式提出关于Unity游戏开发的问题，例如：

- "如何实现Unity中的角色移动系统？"
- "Unity游戏性能优化的方法有哪些？"
- "如何设计一个好的游戏架构？"
- "网络多人游戏的同步机制是什么？"
- "如何在Unity中实现AI寻路？"

### 响应结构

对于每个问题，技能会按照以下结构进行响应：

1. **问题分析**：技术背景、根本原因、解决方案概述
2. **案例演示**：代码示例、实现说明、应用方法
3. **注意事项**：关键点、陷阱、优化建议
4. **实现原理**：底层原理、引擎分析、API介绍
5. **知识点总结**：核心概念、技术要点、学习建议

### 文件管理

所有问题和回答会自动保存为Markdown文件，存储在以下目录结构中：

```
.trae/skills/unity-game-dev-guide/questions/
├── core/             # 核心系统
├── network/          # 网络编程
├── rendering/        # 渲染技术
├── performance/       # 性能优化
├── architecture/      # 架构设计
├── tools/             # 工具链
└── cross-platform/    # 跨平台开发
```

文件命名格式：`YYYY-MM-DD-问题标题.md`
